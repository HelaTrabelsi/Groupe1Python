Gestion Simple de Comptes Bancaires (Tkinter)

Description:

Cette application Python propose une interface graphique complète (Tkinter) pour gérer des comptes bancaires.
Elle permet d’effectuer des opérations classiques comme les dépôts, retraits, transferts, ainsi que la création ou suppression de comptes.

Projet réalisé dans le cadre d’un exercice de programmation orientée objet (POO) avec interface graphique.


Fonctionnalités principales:

Dépôt d’argent sur un compte
Retrait d’argent (avec plafond de retrait)
Transfert d’un compte à un autre
Historique complet des opérations (type, date, montant, solde après opération)
Modification du plafond de retrait
Création / suppression de comptes utilisateurs

Concepts utilisés:

Programmation orientée objet (POO) :
Classe Account (modèle de compte bancaire)
Classe BankApp (application graphique Tkinter)
Interface graphique (Tkinter + ttk)
Gestion d’événements (clics, sélections, saisies)
Structures de données : dictionnaires et listes
Gestion d’erreurs et boîtes de dialogue (messagebox)
